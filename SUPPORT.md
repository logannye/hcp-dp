# Support

HCP-DP is currently an alpha project. The best support path is a GitHub issue
with a minimal reproducer and, when possible, an independent expected result.

Use issues for:

- incorrect score or traceback,
- CLI behavior that is hard to automate,
- documentation gaps,
- reproducible performance regressions,
- requests for additional alignment modes or output formats.

Please include:

- exact command or Rust snippet,
- input sequences or a small input file,
- expected result,
- actual result,
- `hcp-align --version` or the commit SHA,
- operating system and Rust version.

Security-sensitive reports should follow [SECURITY.md](SECURITY.md).
